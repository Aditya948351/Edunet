# Restaurant Menu Creation

## Overview
Created a premium restaurant menu page in `Restaurant menu/index.html`.

## Features
- **Premium Design**: Dark theme with gold accents (`#d4af37`), using 'Playfair Display' and 'Lato' fonts.
- **Responsive Table**: Displays 15 gourmet items with columns for Serial Number, Food Name, Image, Price, and Rating.
- **Animations**: Subtle fade-in animations for the header and table.
- **Hover Effects**: Rows highlight and images scale slightly on hover.
- **Assets**: Generated a sample gourmet food image (`food.png`) used as a placeholder for all items.

## Files
- `Restaurant menu/index.html`: Main HTML file with embedded CSS.
- `Restaurant menu/food.png`: Generated food image.

## Next Steps
